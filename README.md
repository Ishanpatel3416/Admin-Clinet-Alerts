# Client Admin Panel Skeleton

This project provides a basic frontend framework for Bynd's client-admin dashboard. It uses **React**, **TypeScript**, and **Vite** and contains placeholder pages for future integration with backend services.

## Available Sections
- Overview
- Users & Roles
- Limits & Cost Controls
- Domain & SSO
- Billing
- Logs & Exports
- Support

Each page includes minimal structure and `TODO` comments where backend data and actions can be connected later.

## Getting Started
```bash
npm install
npm run dev
```

## Checks
Run the available checks before committing changes:
```bash
npm run lint
npm run build
```
